# xinjiapp

一个面向“智能戒指 + 情感陪伴”方向的 App 原型项目。

当前版本聚焦两个核心页面：
- `设备`：展示戒指连接状态、电量、同步情况，以及已收集到的健康数据
- `设置`：展示账号信息、设备信息、帮助与支持，并支持点开展开详情

## 项目定位

这个原型不是完整的聊天 App，而是一个为 AI 陪伴提供设备与状态上下文的移动端界面。

设计思路：
- App 负责展示设备状态和已采集的用户健康数据
- AI 对话发生在微信等外部渠道
- 界面强调温和、纸感、陪伴型产品氛围，而不是传统硬件后台

## 当前功能

### 设备页
- 戒指状态展示
- 电量与续航信息
- 最近同步、连接质量、固件版本
- 已收集健康数据：
  - 心率
  - 睡眠
  - 步数
  - 经期
- 各项支持点击展开查看详情

### 设置页
- 账号
- 昵称
- 绑定状态
- 数据权限
- 采集频率
- 佩戴指南
- 连接帮助
- 固件更新
- 各项支持点击展开查看详情

## 设计风格

视觉方向基于“心迹档案 / Xinji Pencil App Style”：
- 暖白底色
- 纸感卡片
- 柔和粉、青、米色点缀
- 移动端原型式布局
- 固定手机屏幕高度，内容区域内部滚动

## 本地预览

直接打开：
- `index.html`

或在本地启动静态服务：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173/
```
